# Custom Portrait Tool v1.0.0

初回公開版です。ゲーム用カスタムポートレートを、カテゴリ管理から画像出力までまとめて作成できます。

## 主な機能

- カテゴリ / サブカテゴリごとの背景管理
- 位置ガイド画像の薄表示
- キャラクター画像の切り抜き、拡大縮小、簡易背景透過、左右反転
- ビュー画面での位置調整
- PNG / BMP 出力
- Alpha マスク画像の自動生成
- 固定出力先フォルダへの保存

## 配布ファイル

- `CustomPortraitTool.exe`

## 使い方

1. `CustomPortraitTool.exe` をダウンロードします
2. 任意のフォルダに配置します
3. exe をダブルクリックして起動します

詳しい使い方は README を参照してください。

## 補足

- 設定ファイルや出力画像は、exe を配置したフォルダを基準に作成されます
- Windows の環境によっては初回起動時に警告が表示される場合があります
